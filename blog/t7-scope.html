What 'scope' is, and how it works in JavaScript.

In JavaScript, we can define a veriable and assign values on it. The scope means to what extent we can access this variable. For examle. If variable is outside any function we wrote, we can access it from anywhere. However if it's defind in a function, we can only access it inside this function.
