Q: Explain the difference between accessing data from arrays and objects.
A: Data in arrays are ordered. We can access data by using indexs. Data in objects are not ordered. We can access objects through properties.

Q: Explain what functions are and why they are useful.
A: We put a set of control flow together and give a name. It's become a function. We can call this function whenever we need to do the same thing. It's avoid repeat the same code.
