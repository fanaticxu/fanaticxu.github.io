Feb 1 2018 by Frank

Explain to a non-tech friend

In the JavaScript language, describe all the use cases for each of:

parentheses ()

The formula in the parentheses has the right the calc first. And a function name followed by parentheses means excute this function.

brackets []

We can put a sequence of numbers/strings/arrays/objects into a brackets and it become an array. And if we give array a name, e.g. arr and the arr[0] means the first item in this array and so on and so forth.

braces {}

In JavaScritp braces mostly means scope. For a function, control flow in the braces belong to this function. For a object, all of the arguments in the braces belong to this object.

single quotes ' ' and double quotes " "

In JS they are almost the same. We use quotes to tell computer this is a string. If the string itself contain double quotes, we can use single quotes between it. vice versa.
